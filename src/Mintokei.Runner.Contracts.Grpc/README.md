# Mintokei.Runner.Contracts.Grpc

Generated gRPC service and message contracts for the Mintokei remote-runner protocol.

Use this package when implementing the gRPC data plane between a backend using
`Mintokei.Runner.Host` and a worker using `Mintokei.Runner.Client`.

```bash
dotnet add package Mintokei.Runner.Contracts.Grpc
```

If you only need dependency-free DTOs and tunnel frame codecs, use `Mintokei.Runner.Contracts`.

Part of the **Mintokei Agent Runtime**.
