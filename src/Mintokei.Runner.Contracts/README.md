# Mintokei.Runner.Contracts

Dependency-free wire records and tunnel frame codecs for the Mintokei remote-runner protocol.

Use this package when you need to serialize or inspect runner messages without bringing in gRPC,
ASP.NET Core, EF Core, or the agent engine.

```bash
dotnet add package Mintokei.Runner.Contracts
```

For generated gRPC service/client types, use `Mintokei.Runner.Contracts.Grpc`.

Part of the **Mintokei Agent Runtime**.
